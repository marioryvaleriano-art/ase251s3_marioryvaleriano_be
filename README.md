# Proyecto Backend - Gestión de Cosechas

## Descripción

Este proyecto consiste en la creación de un endpoint utilizando Spring Boot para gestionar información básica de las cosechas de la Bodega Peirano.

## Tecnologías utilizadas

* Java 17
* Spring Boot
* Maven
* IntelliJ IDEA

## Pasos realizados

1. Se creó el proyecto con Spring Initializr.
2. Se configuró el proyecto en IntelliJ IDEA.
3. Se creó el controlador CosechaController.
4. Se implementó un endpoint usando @GetMapping.
5. Se ejecutó la aplicación en el puerto 8080.
6. Se probó el endpoint en el navegador.

## Endpoint de prueba

http://localhost:8080/cosechas
